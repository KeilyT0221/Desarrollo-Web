Introducción

El curso de programación I fundamenta los conocimientos adquiridos en el curso de algoritmos y continúa formando a estudiante con las estructuras de pensamiento necesarias para la carrera. El curso de Programación I enseña los conceptos básicos de programación orientada a objetos, así como también la construcción programas con aplicación a cualquier giro de negocio empresarial. La estructura general de un programa en JAVA y cada una de las distintas instrucciones para la construcción de un programa en JAVA.

Informacion del curso

Este curso está enfocado al diseño de bases de datos, proporcionando al estudiante los conocimientos fundamentales para la comprensión y el diseño de Bases de Datos relacionales. Adicionalmente, se desarrollarán ejercicios con el objetivo de que el estudiante adquiera la habilidad de o de fallos, el manejo de concurrencia, seguridad e integridad de información. Se adiciona al presente programa, la normalización de bases de datos relacionales. Adicionalmente, contempla los sistemas de bases de datos distribuidos, el modelo cliente servidor, orientados a objetos y algunos temas complementarios.

Objetivos

  1. Que el estudiante comprenda los conceptos fundamentales de la programación estructura, procedimental y orienta a objetos que rigen que son base para las buenas prácticas en el desarrollo de software.
  2. Motivar al alumno para crear en él el interés en la utilización de herramientas de programación aplicadas al desarrollo e implementación de sistemas de información.

Alcance del Curso

Al finalizar el curso, el estudiante será capaz de:

  • Explicar los conceptos base necesarios para el desarrollo de aplicaciones.
  • Construir una solución que integre de forma exitosa componentes de interfaz de usuario con componentes de aplicación de base de datos.
  • Integrar una base de datos como componente de las soluciones de aplicaciones.
  • Implementar estructuras de control y datos en sus programas.
  • Reconocer y anticipar los problemas más comunes en el desarrollo de aplicaciones robustas a través del lenguaje de JAVA.
